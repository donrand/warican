# WARICAN - 割り勘アプリ

メンバーの支払いを記録し、誰が誰にいくら払えばよいかを自動計算する割り勘 Web アプリです。

## 機能

- メンバーの登録・削除
- 支払いの記録（支払者・金額・内容・割り勘メンバーの選択）
- 精算結果の自動計算（最小取引数で表示）

## セットアップ

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5174` を開いてください。
devcontainer 環境の場合は VS Code の「ポート転送」タブで `5174` を追加してください。

## ビルド

```bash
npm run build
```

ビルド成果物は `dist/` ディレクトリに出力されます。

## 使い方

1. **メンバー登録** — 参加者の名前を入力して追加
2. **支払い入力** — 支払者・金額・内容・割り勘対象メンバーを選択して追加
3. **精算結果** — 誰が誰にいくら払うかが自動で表示されます
